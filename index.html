Proyectos Destacados

Proyecto de Redes Neuronales para Predicción de Material Particulado

Desarrollo de modelos de redes neuronales, incluyendo LSTM y redes multicapa, para la predicción de niveles de material particulado en el aire. Se evaluaron modelos como:

  • Random Forest: MSE = 732.48, MAE = 13.03, MAPE = 0.55%, R² = 0.73
  • LSTM: Loss = 668.43, MAE = 12.83, MSE = 668.43, R² = 0.75
  • Redes Multicapa: Loss = 814.23, MAE = 13.78, MSE = 814.23, R² = 0.70

El proyecto permitió identificar patrones en la contaminación atmosférica y mejorar la capacidad predictiva para la toma de decisiones en gestión ambiental.
