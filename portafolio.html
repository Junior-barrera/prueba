Proyectos Destacados

Proyecto de Redes Neuronales para Predicción de Material Particulado

Desarrollo de modelos de redes neuronales, incluyendo LSTM y redes multicapa, para la predicción de niveles de material particulado en el aire. Se evaluaron modelos como:

  • Random Forest: MSE = 732.48, MAE = 13.03, MAPE = 0.55%, R² = 0.73
  • LSTM: Loss = 668.43, MAE = 12.83, MSE = 668.43, R² = 0.75
  • Redes Multicapa: Loss = 814.23, MAE = 13.78, MSE = 814.23, R² = 0.70

El proyecto permitió identificar patrones en la contaminación atmosférica y mejorar la capacidad predictiva para la toma de decisiones en gestión ambiental.

Construcción y Evaluación de un Sistema Aeropónico Piramidal Asistido por Arduino para la Producción de Cebollín

Este estudio demuestra la viabilidad de un sistema aeropónico piramidal asistido por Arduino para cultivar cebollín. Se evaluaron variables como el tiempo y frecuencia de riego, y la temperatura interna de las pirámides. El tiempo de riego óptimo fue de 2 minutos, favoreciendo el crecimiento, mientras que la frecuencia de riego y la temperatura no mostraron efectos significativos por sí solas. Sin embargo, su interacción mostró patrones que ayudan a optimizar el crecimiento.

El sistema automatizado con Arduino optimizó el riego y la nutrición, promoviendo un uso eficiente de los recursos, especialmente en un entorno desértico como Arica, Chile. Este enfoque también contribuye a la sostenibilidad en la agricultura, y el sistema piramidal ofrece una solución replicable para el cultivo en climas extremos.
